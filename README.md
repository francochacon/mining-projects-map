# 📊 Superstore Sales Performance Dashboard

An interactive Power BI dashboard built with the Sample Superstore dataset, 
designed to provide executive-level visibility into sales performance, 
profitability, and regional trends.

---

## 🔍 Overview

This dashboard transforms raw retail data into actionable business insights 
through clean visualizations and key performance indicators.

---

## 📈 Key Metrics

| Metric | Value |
|---|---|
| Total Sales | $2.30M |
| Total Profit | $286.4K |
| Profit Margin | 12.47% |
| Total Orders | 5,009 |

---

## 📌 Dashboard Features

- **KPI Cards** — Instant snapshot of Sales, Profit, Margin, and Orders
- **Sales & Profit Trend by Month** — Dual-axis line chart tracking performance across the year
- **Sales by Region** — Horizontal bar chart comparing West, East, Central, and South
- **Top 10 Sub-Categories by Sales** — Ranked view of best-performing product lines

---

## 🛠 Tools & Technologies

- Power BI Desktop
- DAX (calculated measures and columns)
- Power Query (data transformation)
- Dataset: Sample Superstore (Kaggle)
